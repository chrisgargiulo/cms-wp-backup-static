The foundation upon which branding guidelines, visual styles, pattern libraries, and code libraries can be defined and maintained.<\/h3>\n\n\n\n

Design systems are helpful for designers, developers, and content creators to ensure a visual consistency across multiple products from a single brand. They can be small or large and include different types of client deliverables, such as style guides and pattern libraries. Some common elements that might be included within a design system:<\/p>\n\n\n\n

  • Logos (e.g. different versions\/formats\/layouts\/etc)<\/li>
  • Typography<\/li>
  • Colors<\/li>
  • Visual Imagery (e.g. photography, illustration)<\/li>
  • UI Design Patterns (e.g. buttons, form elements, icons, symbols, etc.)<\/li>
  • and much more…<\/li><\/ul>\n\n\n\n
