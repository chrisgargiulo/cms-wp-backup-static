A short intro to the core fundamentals of Javascript for front-end web development<\/h3>\n\n\n\n

Javascript (JS), oftentimes referred to as “Vanilla JS,” is used to added functionality to a web page (e.g. “do this when a user clicks this”). For many beginner web designers, JS can be daunting to learn. Many find JQuery to be much simpler and easier to use, learn, and implement. Knowing that JQuery is a JavaScript library that is based on JavaScript (that comes at the cost of adding it as a required dependency on page load), it makes sense to learn the source language first: JavaScript. Once you understand the basics of Vanilla JS, such as how to write a simple function, then the possibilities become endless. For example, if you can write a simple function that adds and removes a class on an HTML element (DOM manipulation) when a user clicks on something, then you can then manage the UX and control your UI via the HTML and CSS. This core front-end web development skill will enable you to do many, many, many things.<\/p>\n\n\n\n
